# PROJECT STRUCTURE

## Directory Layout
- `src/`: Contains the main application source code.
- `tests/`: Contains unit and integration tests.
- `docs/`: Contains project documentation and guides.
- `assets/`: Contains images, icons, and other static resources.
- `configs/`: Config files for different environments.

## File Organization
- Each module has its own subdirectory under `src/`.
- Test files mirror the structure of the source files for easy navigation, located in the `tests/` directory.
- Documentation files are organized into categories in the `docs/` directory.
